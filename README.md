# world-co2-commission-dashboard

This Power BI dashboard analyzes global CO₂ emissions across different countries and regions. It provides an interactive view of emission trends, country-wise comparisons, and geographical distribution of CO₂ emissions using visualizations such as maps, scatter charts, and pie charts. The dashboard helps users explore emission patterns and identify major contributors to global carbon emissions through easy-to-use filters and interactive reports.

**Tools Used:** Power BI, Power Query
